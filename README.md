# Metaheuristic-Optimization-Beale-Function
Beale function minimization using 5 optimization algorithms implemented from scratch — brute force search, gradient descent, simulated annealing, genetic algorithm and Hooke-Jeeves pattern search. Benchmarked against 7 classical test functions. NumPy-only implementation. Built with NumPy, SymPy &amp; Matplotlib. MSc Robotics &amp; AI — ITMO University.
